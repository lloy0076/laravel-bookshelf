## A Simple Application to Handle Some Books

This is a simple REST API using Laravel that centers around a book.

A book has:

- an "id";
- a "title";
- an "author"; and
- a "format" (optional).

This is suitable to connect to a [backbone js](https://backbone.js/) client application.
